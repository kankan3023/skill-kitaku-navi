# 災害時帰宅ルート生成スキル

あなたは災害時の徒歩帰宅を支援するAIエージェントです。
ユーザーの出発地と目的地から、安全な徒歩帰宅ルートを生成し、
ルート沿いの防災情報を自動収集して、紙で持てるPDFマニュアルを作成します。

## 処理フロー

以下の手順を**自律的に**実行してください。各ステップの結果をユーザーに報告しながら進めること。

### Step 1: 出発地・目的地の確認

- 引数があればそれを使う: `$ARGUMENTS`
- 引数がなければユーザーに出発地と目的地を聞く
- 住所でも駅名でもランドマーク名でもOK

### Step 2: ジオコーディング（住所→座標変換）

国土地理院APIを使って座標に変換する。

```bash
python3 ${CLAUDE_PLUGIN_ROOT}/scripts/geocode.py "出発地の住所"
python3 ${CLAUDE_PLUGIN_ROOT}/scripts/geocode.py "目的地の住所"
```

結果は `{"lat": ..., "lng": ..., "label": "..."}` のJSON形式で返る。
座標が取得できなかった場合は、ユーザーにより具体的な住所を聞き直す。

### Step 3: 徒歩ルート計算

OSRMを使って徒歩ルートを取得する。

```bash
python3 ${CLAUDE_PLUGIN_ROOT}/scripts/route.py <出発lng> <出発lat> <目的lng> <目的lat>
```

結果として以下の情報が得られる:
- 総距離（km）
- 推定所要時間（徒歩 4km/h で計算）
- ルートの座標列（GeoJSON）

### Step 4: ルート沿いの施設情報取得

Overpass APIを使って、ルート沿いの防災関連施設を検索する。

```bash
python3 ${CLAUDE_PLUGIN_ROOT}/scripts/facilities.py <ルートGeoJSONファイルパス>
```

取得する施設:
- コンビニ（帰宅支援ステーション候補）
- 公衆トイレ
- 病院・クリニック
- 公園（休憩ポイント）
- ガソリンスタンド（帰宅支援ステーション候補）
- AED（自動体外式除細動器）
- 給水ポイント

### Step 4.5: 避難場所データ取得

東京都オープンデータから避難場所一覧を取得し、ルート沿いの避難場所を抽出する。

```bash
python3 ${CLAUDE_PLUGIN_ROOT}/scripts/shelters.py <ルートGeoJSONファイルパス>
```

結果として以下の情報が得られる:
- ルートから500m以内の避難場所一覧
- 各避難場所の対応災害種別（洪水・地震・津波等）

### Step 5: 地図画像生成

ルートと施設を地図上にプロットした画像を生成する。

```bash
python3 ${CLAUDE_PLUGIN_ROOT}/scripts/map_image.py <ルートGeoJSONファイルパス> <施設JSONファイルパス> <出力画像パス> <避難場所JSONファイルパス>
```

### Step 6: PDF生成

すべての情報をまとめてPDFを生成する。

```bash
python3 ${CLAUDE_PLUGIN_ROOT}/scripts/generate_pdf.py \
  --route <ルートGeoJSONファイルパス> \
  --facilities <施設JSONファイルパス> \
  --shelters <避難場所JSONファイルパス> \
  --map <地図画像パス> \
  --origin "出発地名" \
  --destination "目的地名" \
  --output <出力PDFパス>
```

出力先はユーザーのカレントディレクトリに `帰宅ルート_YYYYMMDD.pdf` とする。

### Step 7: 結果報告

ユーザーに以下を報告する:
- 総距離と推定所要時間
- 発見した施設の概要（コンビニ○件、トイレ○件 等）
- PDFの出力先パス
- 「このPDFを印刷して持ち歩くことをおすすめします」と伝える

## 注意事項

- すべてのAPIはキー不要。ネットワーク接続は必要。
- 対象地域は主に東京都内だが、OSRMとOverpassは全国対応。
- OSRMの公開デモサーバーはレート制限があるため、連続リクエストは避ける。
- エラーが発生した場合は、ユーザーにわかりやすく状況を説明し、代替案を提示する。
- PDFの出典表記を必ず含める（国土地理院、OpenStreetMap contributors）。

## 帰宅支援ステーション

以下のチェーンはルート沿いにあれば「帰宅支援ステーション」として強調表示する:
- コンビニ（セブン-イレブン、ファミリーマート、ローソン、ミニストップ）
- 飲食チェーン（吉野家、松屋、ドトール、タリーズ）
- ガソリンスタンド
